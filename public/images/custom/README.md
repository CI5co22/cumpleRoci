Usa esta carpeta para tus imagenes personalizadas.

Nombres disponibles:
- `img1` = primer recuerdo que aparece al tocar la primera estrella
- `img2` = segundo recuerdo
- `img3` = tercer recuerdo
- `img4` = cuarto recuerdo
- `img5` = quinto recuerdo
- `img6` = sexto recuerdo extra si siguen tocando estrellas
- `img7` = foto circular final del modal de cumple

Extensiones soportadas por el programa:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`
- `.gif`
- `.svg`

Ejemplo:
- `img1.jpg`
- `img2.png`
- `img7.webp`

Si existen varias extensiones con el mismo nombre base, el programa intenta en este orden:
`jpg`, `jpeg`, `png`, `webp`, `gif`, `svg`
